## How to Use This Template

1. Copy this file to `~/.xorchestrator/workflows/`
2. Rename it to reflect your workflow (e.g., `code_review.md`)
3. Update the frontmatter:
   - `name`: Display name shown in the picker
   - `description`: Brief description shown below the name
   - `category`: Optional grouping (for future use)
4. Customize the content with your workflow instructions
5. Open orchestration mode and press `Ctrl+P` to select your workflow
